# Premium Online Multiplayer Chinese Checkers 🎮

A real-time, modern, and completely mobile-responsive online multiplayer Chinese Checkers game. Built from scratch using HTML5 Canvas to ensure smooth gameplay, precise touch controls, and zero UI glitching or zooming on mobile devices.

## ✨ Features

* **Real-Time Multiplayer:** Play instantly with friends using room codes powered by Socket.io.
* **Flawless Mobile Experience:** Custom touch-coordinate mapping and absolute no-zoom enforcement for perfect mobile gameplay.
* **HTML5 Canvas Rendering:** The game board and marbles are drawn on a canvas, completely eliminating CSS scaling bugs and layout shifts.
* **Premium Aesthetic:** Sleek, dark-mode UI designed for an immersive and modern feel.
* **Fast-Paced Rules:** Enjoy dynamic and quick movements with multi-jump logic.
* **Auto-Reconnect:** Built-in network resilience for seamless mobile network transitions.

## 🛠️ Tech Stack

**Frontend:**
* HTML5 Canvas API
* Vanilla JavaScript
* CSS3

**Backend:**
* Node.js
* Express
* Socket.io
